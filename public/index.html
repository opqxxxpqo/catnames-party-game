      玩家

      Handbook

      完整规则

      棋盘

      5×5 共 25 张喵卡，每张有隐藏身份。身份分布按当前玩法模式不同。

      身份图例

      • 目标 / 队伍色：你要引导队友翻到的喵。
      • 路过：翻到即结束回合。
      • 危险：翻到扣机会。
      • 失败：翻到直接判负。

      提示格式

      一个词 + 一个数字（0–9）。同一条提示最多可猜 数字 + 1 次（奖励猜）。例："动物 3" 可猜最多 4 张。

      提示词合法性

      经典模式生效，更多喵 · 纯图模式跳过：

      • 必须是单个词，不能有空格
      • 不能纯数字
      • 不能混用中英文
      • 不能包含棋盘上任何未翻开的喵名里的字符（比如棋盘有"迷路喵"就不能用含"迷"或"路"的词）

      图片模式（与玩法模式正交）

      模式图源呈现
      经典 · 配字版63 只本地 HTTP 喵图 + 中文喵名 + 描述
      更多喵 · 纯图实时从 CATAAS 抓 25 张纯图 + #序号

      答案卡（右侧小棋盘）

      只对"能看身份"的人显示：2 人都看自己那份密钥；3–5 人只有当前提示者看；6–8 人只有 Spymaster 看。纯图模式下只显身份色 + 序号。

      2 人合作解谜 (duet_coop)

      触发：2 人。招牌体验："你居然猜到了我想说的意思。"

      身份分布

      两套独立密钥 A / B，各 9 目标 / 13 路过 / 2 危险 / 1 失败。同一张卡可能 A 视角是目标、B 视角是路过。

      流程

      1. A 给 B 出提示，B 按 A 的密钥翻牌
      2. B 可连续翻，直到 count+1 用完 / 猜错 / 点"结束回合"
      3. 翻到路过 / 危险 → 回合立即结束
      4. 角色互换，换 B 给 A 出提示
      5. 最多 9 回合；累计 3 次危险 或 1 次失败立刻判负

      胜负

      赢：两套密钥里所有目标（并集）都被找到。
      输：9 回合用尽 / 3 次危险 / 翻到失败喵。

      计分（MVP 展示）

      • 提示者：命中 +2，危险 −1，失败 −3
      • 猜测者：命中 +1，危险 −1

      3–5 人半合作同步揭示 (semi_coop)

      触发：3–4 人强制；5–6 人房主可选。招牌体验："我们同时点了老虎！"——默契命中的爆点。

      身份分布

      一套公共密钥：9 目标 / 13 路过 / 2 危险 / 1 失败，只有当前提示者能看身份。

      阶段状态机（全程硬时限，超时自动推进）

      1. clue（60s）：提示者输入词 + 数字，guessesLeft = count + 1
      2. secretSelect（30s）：所有猜测者同时秘密各选一张
      3. 同步揭示三种结果：
        全员一致直接翻牌，提示者额外 +1 默契分，命中者分数 ×2
        有分歧进 discuss(30s) → 提示者按"开启投票" → vote(20s) → 提示者结算多数票
        完全分歧进 decide(20s)，提示者只能从猜测者实际选过的那几张里挑一张
      4. 翻牌结算：
        • 目标：消耗一次猜测机会；guessesLeft > 0 → quickContinue(20s)（投票继续 / 收手）；= 0 → 结束回合
        • 路过 / 危险 / 失败：立刻结束回合
      5. 回合结束 → 提示者轮到下一位

      全局硬时限

      单局最长 20 分钟，时间到按当前进度判负。

      胜负

      赢：翻完全部 9 张目标，保留个人分做 MVP 排行。
      输：失败喵 / 危险累计 3 次 / 20 分钟用尽 → 团队失败，所有个人分清零（防止队友在必败局刷分）。

      计分

      • 提示者：命中 +2，全员默契再 +1，危险 −1，失败 −3
      • 猜测者：命中 +1，全员默契再 +1（翻倍），危险 −1

      6–8 人阵营对抗 (team_vs_team)

      触发：7–8 人强制；5–6 人房主可选。招牌体验："绝杀！"——最后一张抢在对面手里翻回来。

      身份分布

      先手队 9 张 / 后手队 8 张 / 7 路过 / 1 失败。按加入顺序偶/奇分红蓝，每队前两位默认 Spymaster。

      流程

      1. 当前队 Spymaster 出提示（词 + 数字）
      2. 本队进入 reveal 阶段，队员自由点牌（无秘密投票）
      3. 每翻一张按身份结算：
        翻到结果
        自己队的喵+2 分，guessesLeft--；用完 → 换队；未用完 → 可继续翻
        对方队的喵帮对面翻了一张，Spymaster −1，回合结束
        路过喵回合结束
        失败喵对方队直接获胜
      4. 队员或 Spymaster 都能按"结束回合"主动收手

      胜负

      赢：把本队全部喵翻完（先手 9 / 后手 8）。
      输：对方先翻完 / 自己翻到失败喵。

      计分

      • Spymaster 每命中 +2，翻错帮对方 −1。
